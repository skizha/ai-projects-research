# AI Projects Research

A collection of guides covering production AI system architecture, real-world project case studies, and financial AI applications.

## Contents

### [AI Projects Architecture Guide](./ai-projects-architecture-guide.md)
Five interconnected projects that build progressively sophisticated AI systems — covering the full lifecycle of production AI development: RAG pipelines, local optimization, observability, model customization, and real-time systems.

### [Financial AI Systems Guide](./financial-ai-systems-guide.md)
Real-world applications of the 5-project architecture in finance — from earnings report analysis and competitive intelligence through algorithmic trading systems.

### [Real-World Project Examples](./real-world-project-examples.md)
Production case studies across industries including LegalTech, enterprise search, and document intelligence — showing how the architecture patterns apply in practice.
